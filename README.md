# MicroJava Compiler

A compiler for the MicroJava programming language implemented in Java.

## Features
- Lexical analysis using **JFlex**
- Syntax analysis using **Java CUP**
- Semantic analysis with symbol table
- Bytecode generation for the MicroJava virtual machine

## Supported Language Constructs
- Variables and constants
- Arrays
- Functions
- if / else statements
- for loops
- ternary operator
- switch statement

## Technologies
- Java
- JFlex
- Java CUP
- MicroJava VM
